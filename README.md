# 🎮 Reino Matemático

## 📌 Sobre o Projeto

O **Reino Matemático** é um serious game desenvolvido como Trabalho de Conclusão de Curso (TCC), com o objetivo de apoiar o ensino de matemática para estudantes portadores de TDAH nos anos finais do Ensino Fundamental I.

O jogo utiliza elementos de RPG para transformar o aprendizado em uma experiência interativa, envolvente e acessível, especialmente considerando alunos com TDAH. A proposta combina narrativa, desafios matemáticos e mecânicas de jogo para estimular o raciocínio lógico de forma lúdica.

Se preferir jogar localmente:

1. Clone este repositório:


2. Acesse a pasta do projeto

3. Execute o jogo conforme as instruções do projeto (dependendo da build exportada)

## 🎯 Objetivos do Jogo

* Auxiliar no aprendizado de operações matemáticas básicas
* Estimular o raciocínio lógico
* Promover engajamento através da gamificação
* Oferecer uma experiência acessível para alunos com TDAH

---

## 🧠 Tecnologias Utilizadas

* RPG Maker MV
* Design de jogos educacionais (serious games)
* Metodologias baseadas na BNCC
